      이 두 TAG는 서로 경쟁관계이다.
      <br>는 줄바꿈으로 표현하며 시각적인 의미만 있기 때문에 닫을 필요가 없어요!

      <p>는 단락이라는 의미로 br태그와는 다르게 어디서부터 어디까지 한 단락인지를 표현해주기 때문에 태그를 열고 닫아줍니다.

      💡 CSS의 margin 속성은?
      원하는 단락구간에 <p style="margin-top:40px;">를 넣어줍니다.

  2. 속성 Tag를 알아볼까요?
    • img태그는 <img src"">예요
      이미지는 언스플래시 < 클릭해주세요.

      이미지와 연관된 속성을 알아볼까요?
      1. width는 폭,너비를 나타내요.
      2. height는 높이를 나타내요.
      3. border: 3px 테두리의 두께를 말해요
      4. solid : 색깔은 테두리의 색을 말해요
      5. border-radius는 테두리를 둥굴게 해줘요
        6.    - 웹브라우저 마다 설정이 필요해요
             ex) moz(모질라), khtml,webkit(chrom,safari),ms,o(오페라)

  3. 부모 자식과 목록은 무엇일까요?
    • ul, ol는 부모태그 li는 자식태그를 말해요
    • ul는 기호리스트, ol는 숫자리스트입니다

  4. 테이블에 대해 알아볼까요?

    태그 안내
    테이블의 구성 <tabel> 테이블을 만들어요
    <th> 테이블의 헤더부분을 만들어요
    <tr> 테이블의 행을 만들어요
    <td> 테이블의 열을 만들어요

    속성 안내
    테이블의 디자인 <border> 테두리를 설정해요
    <bordercolor> 테두리의 색상을 설정해요
    <width> 테두리의 가로 크기를 설정해요
    <height> 테두리의 세로 크기를 설정해요
    <align> 정렬을 의미해요
    <bgcolor> 배경색을 의미해요
    <colspan> : 가로(열) 합병을 의미해요>
    <rowspan> 세로(행) 합병을 의미해요
